## Compile

You need to install :
* [libcaca](http://caca.zoy.org/wiki/libcaca)
* [SDL 2](https://www.libsdl.org/)
* [SDL_mixer 2](https://www.libsdl.org/projects/SDL_mixer/)

Then :

```
make
```

## Run

```
./tetris
```
